Este proyecto es una aplicación web desarrollada con HTML, CSS y JavaScript que permite gestionar una agenda de contactos de forma dinámica mediante el uso de la función fetch() para consumir una API externa. La aplicación obtiene los contactos almacenados utilizando el método HTTP GET desde http://www.raydelto.org/agenda.php y los muestra en una tabla con diseño corporativo tipo dashboard; además, permite agregar nuevos contactos enviando datos en formato JSON mediante el método HTTP POST a la misma URL. La interfaz fue diseñada con un estilo profesional y empresarial, utilizando una estructura limpia, responsive y organizada para simular un sistema real de gestión de contactos.
